Wanderlust Adventures Website Project
====================================

Project Description:
A responsive multi-page travel website built with HTML, CSS, and JavaScript for the TECH4200 Website Development assessment.

Features:
- 5 interconnected pages (Home, Europe, Asia, Travel Tips, Contact)
- Responsive design that works on all devices
- Interactive image slider on homepage
- Form validation on contact page
- Consistent navigation across all pages
- Modern, visually appealing design

File Structure:
- index.html       - Homepage
- europe.html      - Europe destinations
- asia.html        - Asia destinations
- tips.html        - Travel tips page
- contact.html     - Contact form page
- css/style.css    - Main stylesheet
- js/script.js     - Main JavaScript file
- images/          - All image assets

Assessment Requirements Met:
- Minimum 5 interconnected pages ✓
- CSS styling for visual appeal ✓
- JavaScript interactivity ✓
- Responsive design ✓
- Debugged functionality ✓
